## taro

> https://nervjs.github.io/taro/docs/GETTING-STARTED.html

### 初体验

* 安装
```
    npm install -g @tarojs/cli

    npm install -g babel-cli 
    npm install -g babel-preset-es2015
```

* 新建项目
```
    taro init test-01
```

* 启动h5
```
    # npm script 
    npm run dev:h5 
    npm run build:h5
```

* 微信⼩程序端
```
    # npm script 
    npm run dev:weapp 
    npm run build:weapp
```
* 浏览器端演示
```
    # npm script 
    npm run dev:h5 
    npm run build:h5
```
* 百度⼩程序端
```
    npm run dev:swan
```
